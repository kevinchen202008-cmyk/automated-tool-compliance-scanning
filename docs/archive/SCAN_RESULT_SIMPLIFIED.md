# 扫描结果简化更新

## 更新内容

根据用户需求，已简化扫描结果展示，**移除了所有评分信息**，只重点呈现以下4个关键信息：

### 1. 📄 使用许可/开源协议
- 许可证类型（MIT、Apache、GPL、商业许可证等）
- 许可证版本号
- 许可证模式（开源/商业/混合）

### 2. 🏢 公司信息（标注是否有中国服务）
- 公司名称
- 所属国家/地区
- 公司总部所在地
- **中国服务状态**（✓ 有中国分公司或服务 / ⚠ 无中国分公司或服务）

### 3. 💼 商用用户使用限制（高亮显示）
- 需购买License：是/否（高亮显示）
- 允许免费商用：是/否（高亮显示）
- 限制说明（用户数、功能、服务器等）
- 用户限制
- 功能限制

**显示样式**：
- 需要购买license：黄色背景警告卡片
- 允许免费商用：绿色背景安全卡片

### 4. 🔄 可替代方案（只显示1-2个）
- 替代工具名称
- 类型（开源/免费商业）
- 许可证类型
- **优势**（重点说明为什么适合替代）
- 适用场景
- 链接（如有）

**限制**：只显示前2个最合适的替代方案

## 已移除的内容

- ❌ 综合评分
- ❌ 各维度评分（安全性、许可证、维护性、性能、TOS合规）
- ❌ 合规状态标识（合规/不合规）

## 报告结构

简化后的报告结构：

```json
{
    "tool": {
        "name": "工具名称",
        "version": "版本"
    },
    "license_info": { ... },            // ⭐ 重点信息
    "company_info": { ... },            // ⭐ 重点信息
    "commercial_restrictions": { ... }, // ⭐ 重点信息（高亮）
    "alternative_tools": [ ... ]         // ⭐ 重点信息（1-2个）
}
```

## Web UI 展示

扫描结果现在以清晰的卡片形式展示：

1. **工具名称**（顶部）
2. **使用许可/开源协议**（灰色卡片）
3. **公司信息**（灰色卡片，标注中国服务状态）
4. **商用用户使用限制**（黄色/绿色高亮卡片）
5. **可替代方案**（蓝色卡片，最多2个）
